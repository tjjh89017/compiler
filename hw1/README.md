#Compiler HW1

##Requirement

- Unix-like OS
- Python3

##1. NFA to DFA

Execute NFAtoDFA.py by followed command

```
./NFAtoDFA.py input.txt output.txt
```

where `input.txt` is NFA and `output.txt` is output DFA.

##2. DFA

Execute DFA.py by followed command

```
./DFA.py input.txt
```

and then input the string by keyboard
